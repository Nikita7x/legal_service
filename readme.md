# Требования
- Docker
- Docker-compose

# Запуск
```bash
docker compose up
```
Сервис будет доступен по адресу localhost:8000

Эндпоинт: 
```curl
http://localhost:8000/parse/
```

Формат данных:
```
{ "data": ТЕСТОВЫЕ ДАННЫЕ }
```
или
```
<root>
  <data>
  ТЕСТОВЫЕ ДАННЫЕ
  </data>
</root>
```
